Структура:

reelance_platform/
│
├── app/	Основное приложение
│   ├── __init__.py
│   ├── main.py		Главный файл приложения, который запускает сервер
│
├── models/	работа с данными и БД
│   ├── __init__.py
│   ├── database.py	Настройка подключения к базе данных
│   ├── models.py		Определяет модели данных (структура таблицы БД) с помощью ORM (например, SQLAlchemy)
│   └── schemas.py		Содержит Pydantic-схемы для валидации данных (проверка)
│
├── routes/	Содержит маршруты (эндпоинты) API/веб-приложения
│   ├── __init__.py
│   ├── auth.py		Роуты для аутентификации: вход, регистрация, выход (Вход пользователей и админов)    
│   ├── admin.py		Роуты для администратора: управление (Админ-панель Добавление/удаление вакансий)
│   ├── jobs.py		Роуты для работы с заданиями (Обработка откликов на вакансии)       
│   └── users.py		Роуты для работы с пользователями (Регистрация новых пользователей) 
│
├── templates/             
│   ├── base.html           
│   ├── index.html		Главная страница сайта        
│   │
│   ├── admin/              
│   │   ├── dashboard.html		админ-панель
│   │   └── login.html			форма входа для администраторов
│   │
│   └── user/              
│       ├── login.html			форма входа     
│       └── register.html		форма регистрации  
│
├── static/                 
│   ├── styles.css		сс     
│
├── .env		окружение            
└── README.md
